# GRRMOD
GRRMOD is a MOD player library for Wii application developers.

It supports the following format:
 * 669 (Composer 669)
 * MOD (15 and 31 instruments)
 * MED (OctaMED)
 * MTM (MultiTracker Module editor)
 * IT (Impulse Tracker)
 * S3M (Scream Tracker 3)
 * XM (FastTracker 2)
